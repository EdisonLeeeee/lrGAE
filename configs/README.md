# Supplemental YAML Configs

This bundle contains clean YAML configuration files for the supplemental baselines.

## Directory layout

- `node_classification/`
- `link_prediction/`
- `graph_clustering/`
- `graph_classification/`
- `heterogeneous_node_classification/`

Each YAML file contains only the following keys:

- `task`
- `method`
- `data_root`
- `supported_datasets`
- `base_config`

A summary index is provided in `config_summary.csv`.
